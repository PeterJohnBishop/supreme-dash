# supreme-dash

# Identity Service

Node.js/Express server with a GraphQL API for user management in MongoDB.
- Externally at http://localhost/graphql/ 
- Internally at http://identity-service:4000/graphql/ 

# Websocket Service 

Gin server with Websocket connection and REST API
- Externally at http://localhost/ws/ and http://localhost/api/
- Internally at http://gin-server:8080/ws and http://gin-server:8080/api
